# NET-COMM
Repository to store networks and communication basic practices.
## Use
- You can use the **make** command to generate the executables and run them in different terminals to test the program.
- Use *./server.exe* and *./client.exe* to run the executables (unless you've changed their names).
### Nota
This program is designed for one server and multiple clients depending on the version.
